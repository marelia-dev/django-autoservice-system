# Django AutoService System

Šis repozitorijus skirtas praktinėms užduotims, atliktoms Django kurso metu.

### Apie projektą

Tai savarankiškos praktinės užduotys, skirtos įtvirtinti Django framework'o žinias.  
Projektas simuliuoja autoserviso užsakymų valdymo sistemą.

### Projekto eiga

Projektas buvo kuriamas savarankiškai po kiekvienos pamokos:

- Modelių kūrimas (Automobilis, Užsakymas, Atsiliepimas, Profilis)
- Vartotojų registracija ir autentifikacija
- CRUD operacijos su Class-Based Views (ListView, CreateView, UpdateView, DeleteView)
- Darbas su nuotraukomis (automatinis apkarpymas ir sumažinimas)
- Atsiliepimų sistema prie užsakymų
- Paieška pagal automobilio duomenis
- Kalbų palaikymas (LT, EN, RU)
- Bootstrap 5 dizainas

### Naudotos technologijos

- Python
- Django 6.0
- Bootstrap 5
- SQLite
- TinyMCE
- Crispy Forms
- Pillow

### Įgyvendintos funkcijos

- Vartotojų registracija ir prisijungimas
- Užsakymų kūrimas, redagavimas ir trynimas
- Tik savo užsakymų matymas
- Nuotraukų pridėjimas ir apdorojimas
- Atsiliepimų palikimas
- Paieška ir filtravimas

### Tikslas

Praktinis Django įgūdžių lavinimas: darbą su modeliais, Class-Based Views, formomis, autentifikacija ir vartotojo sąsaja.

### Pastaba

Šis projektas yra mokomosios užduotys, atliktos Django kurso metu.
